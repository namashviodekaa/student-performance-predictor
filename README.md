# 🎓 Student Performance Predictor (ML-BYOP)

### Project Overview
This project is a **Bring Your Own Project (BYOP)** capstone for the Machine Learning & AI course. It features an interactive Web Application that predicts a student's final exam score based on their daily habits. 

The core of the application uses **Supervised Learning (Linear Regression)** to find the mathematical relationship between study effort, attendance, and academic success.

### 📚 Module Summaries (Course Integration)
This project directly applies the following concepts from the course syllabus:

| Module | Application in Project |
| :--- | :--- |
